# 💊 Pharmacy Management System

A comprehensive Pharmacy Management System designed to streamline pharmacy operations, inventory tracking, supplier management, user administration, and sales processes. The system consists of a modern frontend and a powerful backend API that work together to provide an efficient pharmacy management experience.

---

## 📌 Overview

The Pharmacy Management System helps pharmacies manage their daily operations digitally by providing tools for:

- Inventory & Stock Management
- Medicine Expiration Tracking
- Supplier & Wholesaler Management
- Customer Management
- User & Role Management
- Sales & Purchase Tracking
- Dashboard & Analytics
- Secure Authentication & Authorization

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/pharmacy-management-system.git
cd pharmacy-management-system
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file in the backend directory.

```env
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```
